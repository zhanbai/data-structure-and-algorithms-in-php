# 数据结构与算法 PHP 语言描述

《数据结构与算法 PHP 语言描述》是一本由展小白编写，旨在通过 PHP 语言来描述、学习数据结构与算法的自由图书。如果你没有接受过正规的计算机科学教育，想要学习数据结构与算法，那这本书就是为你准备的。
